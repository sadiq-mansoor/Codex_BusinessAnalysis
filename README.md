## 🎯 The brief

CodeX's marketing team needed to understand the Indian energy-drink consumer before scaling. This
project analyzes the market to answer:

- **Who** are the target consumers (demographics, preferences, consumption habits)?
- **What** drives brand choice — price, taste, ingredients, marketing?
- **Where** should marketing spend go for the strongest penetration?
- **How** does CodeX compare against established competitors?

## 🔍 Approach

1. **Excel** — cleaned and structured the survey responses; built the insight base
2. **Power BI** — an interactive report communicating findings to a marketing audience
3. **Synthesis** — translated the numbers into concrete marketing recommendations
